🔌 การเชื่อมต่อฐานข้อมูลหลัก (Firebase Firestore)

ระบบนี้ใช้ Firebase Firestore เป็นฐานข้อมูลหลักในการจัดการประวัติการส่งงานและรายชื่อผู้ใช้งานทั้งหมด

* คัดลอกจาก Firebase Console > Project Settings > Web App SDK config

กำลังโหลดผู้ใช้งาน...

📢 การแจ้งเตือนผ่าน Telegram Bot

ตั้งค่าส่งข้อความแจ้งเตือนอัตโนมัติไปยัง Telegram Chat / Group เมื่อมีช่างส่งงานใหม่เข้ามาสำเร็จ

* ขอสร้างบอทได้จาก @BotFather ใน Telegram แล้วคัดลอก HTTP API Token มาใส่ที่นี่
* หา Chat ID ได้จากบอทระบุ ID (เช่น @userinfobot หรือ @RawDataBot สำหรับกลุ่ม)

📁 โฟลเดอร์จัดเก็บแยกตามประเภทงาน

กำหนด Google Drive Folder ID ในการอัปโหลดไฟล์เก็บแยกประเภทงาน

🛡️ จำกัดขนาดไฟล์อัปโหลดส่งงานสูงสุด

💾 กู้คืนระบบฐานข้อมูล (Restore Database Options)

นำเข้าข้อมูลสแนปชอตเพื่อกู้คืนข้อมูลคงคลังและรายการประวัติ

📂 กู้คืนจากไฟล์เครื่อง (.JSON File Import)

อัปโหลดไฟล์สำรองข้อมูลสกุล `.json` ที่คุณดาวน์โหลดออกจากระบบไว้ก่อนหน้า ระบบจะทำการตรวจสอบและนำเข้าข้อมูลประวัติทับสต็อกเดิมทั้งหมด

☁️ คลิก หรือลากวางไฟล์สำรองที่นี่ รองรับไฟล์เฉพาะประเภท .json เท่านั้น

☁️ ไฟล์สำรองข้อมูลบน Google Drive จริง

เชื่อมต่อบัญชี Google Drive เพื่อบันทึกข้อมูลสำรองขึ้นระบบคลาวด์โดยตรงอย่างปลอดภัย

🟢 เชื่อมต่ออยู่: user@gmail.com

รายการไฟล์สำรองข้อมูลจริงในโฟลเดอร์ Coway_Stock_Backups บน Google Drive:

ไม่พบไฟล์สำรองข้อมูลบนโฟลเดอร์ Google Drive
